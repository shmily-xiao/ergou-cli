# Ergou CLI 项目概述

## 🎯 项目愿景

创建一个**通用的、开源的**大模型终端 Agent 工具，支持多厂商大模型，具备完整的代码理解和生成能力，让开发者能够自由选择最适合的 AI 模型。

## 📊 项目背景

### 问题陈述

当前 Claude Code CLI 存在以下限制：
1. **绑定单一厂商** - 只能使用 Anthropic 的 Claude 模型
2. **成本较高** - 无法使用性价比更高的国产模型
3. **网络限制** - 国际 API 在国内访问不稳定
4. **无法定制** - 闭源项目，无法根据需求定制功能

### 解决方案

Ergou CLI 提供：
1. **多模型支持** - 一套代码，支持所有主流大模型
2. **成本优化** - 自由选择性价比最优的模型
3. **本地优化** - 针对国内网络环境优化
4. **完全开源** - MIT 许可证，可自由定制和扩展

## 🎨 核心特性

### 1. 多模型支持
- ✅ 阿里云 Qwen 系列
- 🚧 Anthropic Claude 系列
- 🚧 DeepSeek 系列
- 🚧 OpenAI GPT 系列
- 🚧 Moonshot Kimi 系列
- 🚧 智谱 GLM 系列

### 2. 完整工具链
- 文件操作（读/写/编辑）
- Shell 命令执行
- Git 集成
- 代码搜索（grep/glob）
- MCP 协议支持
- 网络搜索
- 子 Agent 调用

### 3. Agent 能力
- 任务分解和规划
- 多轮对话管理
- 工具自动调用
- 上下文窗口管理
- 记忆系统（短期/长期）

### 4. 用户体验
- 流式输出
- 语法高亮
- Markdown 渲染
- TUI 界面（Ink + React）
- 实时成本和 Token 显示

## 🔐 安全考虑

1. **命令执行沙箱** - 危险操作需要确认
2. **文件系统权限** - 限制敏感目录访问
3. **API Key 加密** - 安全存储凭证
4. **敏感信息脱敏** - 日志自动脱敏
5. **审计日志** - 所有操作可追溯
